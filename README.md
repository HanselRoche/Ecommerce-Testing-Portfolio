ECOMMERCE TESTING PORTFOLIO
Author: Hansel Roche

OVERVIEW
This repository contains manual test cases, bug reports, and execution evidence for the Swag Labs (SauceDemo) e-commerce environment.

CONTENTS
1. Hansel_Roche_Test_Suite.xlsx
   - A comprehensive test suite covering Login, Inventory, Cart, Checkout, and Navigation modules.
   - Includes 25 executed test cases with Pass/Fail status and actual results.

2. Defect Reports (Bug Documentation)
   - Defect_Report_TC002.txt: Input Validation bug (Zip Code field accepts special characters).
   - Defect_Report_TC012.txt: Visual/UI bug (Incorrect product image mapping for 'problem_user').
   - Defect_Report_TC018.txt: State Logic bug ('Reset App State' fails to update UI buttons).
   - Defect_Report_TC020.txt: Business Logic bug (System allows checkout with an empty cart).

3. Evidence Screenshots
   - Evidence_Visual_Bug.png: Proof of improper image mapping for the 'problem_user' role.
   - Evidence_Console_Error.png: Browser console logs capturing client-side errors.

TOOLS USED
- Manual Testing
- Brave Browser Developer Tools (Console/Network inspection)
